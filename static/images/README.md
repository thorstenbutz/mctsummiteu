# Site images

- **avatar.png** — Profile avatar used by the GitHub Style theme when no `avatar` param is set. Add or replace this file with your image; keep the filename `avatar.png` so the theme finds it.
- **Light/dark avatar** — In `hugo.toml`, set `avatar` (light mode) and `avatarDark` (dark mode) to different image paths (e.g. `/images/logo-light.png` and `/images/logo-dark.png`) so the profile picture flips with the theme toggle.
